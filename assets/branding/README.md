# MioDecor brand kit (Homey SDK focus)

## Style direction
Minimalist + technical, modern, with a premium-but-friendly feel. Primary usage is within the Homey app UI.

## Palette (decent, modern)
- Charcoal: `#111111`
- Slate: `#1C1C1C`
- Cool gray: `#2B2B2B`
- Accent blue: `#1F6FEB`
- Light surface: `#F5F7FA`
- White: `#FFFFFF`

## Typography (sans-serif)
- Primary: **Inter** (clean, technical, widely available)
- Alternate: **Manrope** (friendly geometric)
- UI-safe fallback: `system-ui, -apple-system, Segoe UI, Roboto, sans-serif`

## Slogan options
1. Smart. Modern. Human.
2. Modern tech, human feel.
3. Smart comfort. Human control.
4. Minimal tech. Maximum feel.
5. Tech that feels right.
6. Modern living, human touch.

## Logo concepts (SVG)
- **Concept A – Shield Circuit**: minimal shield outline with a circuit line and node. Use for a secure, technical tone.
- **Concept B – Hex Core**: hexagon frame with an inner core dot to reference smart-home nodes.
- **Concept C – Monoline M**: monoline “M” built from a single stroke + node, playful yet premium.

Files:
- `concept-a-shield-circuit.svg`
- `concept-b-hex-core.svg`
- `concept-c-monoline-m.svg`

## Usage guidance (Homey)
- Use icon-only SVG for small sizes (app icon, tiles).
- Use wordmark + icon for store listing and marketing.
- Prefer dark background with blue accent for premium/tech feel.
